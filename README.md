# FP&A Monthly Reporting Project

This project demonstrates a typical FP&A monthly reporting workflow for a large bank.

## Project Overview
This analysis replicates a financial planning and analysis process including:

• Budget vs Actual comparison  
• Forecast tracking  
• Variance analysis  
• KPI performance monitoring  
• Executive reporting  

The case study uses an illustrative JPMorgan-style dataset.

## Files

Financial_Model.xlsx
Contains:
- Actual vs Budget vs Forecast analysis
- Variance calculations
- KPI dashboard

Executive_Readout.pptx
5-slide executive summary covering:
- Financial performance
- Variance drivers
- KPI review
- Business outlook
